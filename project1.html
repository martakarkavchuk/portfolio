Mini-Projects and Timeline

As part of my learning process, I plan to complete mini-projects every 3 weeks to apply my knowledge and strengthen my understanding. Below are the mini-projects I plan to work on:

Mini-Project 1: Setting Up a Small Office/Home Office (SOHO) Network

Timeline: Week 12

Description: The first project will focus on setting up a simple, secure, and functional SOHO network. I'll create a basic network structure that supports wireless and wired devices, ensuring a secure connection using basic networking protocols (TCP/IP, DHCP, DNS). I’ll also apply knowledge of subnets and IP addressing to optimize network performance and security.

Mini-Project 2: Creating and Configuring VLANs

Timeline: Weeks 15-16

Description: The second mini-project will involve the configuration of Virtual Local Area Networks (VLANs). I’ll learn how to logically segment a network to improve performance and security. Each VLAN will be designed to group devices with similar needs and work on inter-VLAN routing to ensure proper communication between the segments.

By completing these mini-projects, I’ll not only reinforce my knowledge of network concepts but also gain hands-on experience with real-world networking and cybersecurity techniques.
